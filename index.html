Benchmark overview

Causal reasoning and compositional reasoning are two core aspirations in generative AI. Measuring the extent of these behaviors requires principled evaluation methods. We explore a unified perspective that considers both behaviors simultaneously, termed compositional causal reasoning (CCR): the ability to infer how causal measures compose and, equivalently, how causal quantities propagate through graphs.

Disentangling signal from noise in reasoning evaluation

  • Reasoning vs training data recall. Completely fictitious world models help prevent the conflation of recall and reasoning. Variables and their causal relations have no real-world counterparts, ensuring that they were never seen by the model during training.
  • Reasoning vs in-context recall./b> Compositional consistency evaluation helps disentangle in-context recall (e.g., extraction of relations directly from the context prompt) from reasoning (e.g., correctly inferring distal causal relationships that were never directly stated in the context prompt).
  • Causal reasoning vs numeracy. This benchmark contains both quantitative and qualitative CCR tasks (i.e., those requiring and not requiring numerical understanding, such as ordering numbers by magnitude). If an LM does significantly better on qualitative tasks, this could indicate poor numeracy as a root of failure on quantitative CCR tasks.

Three levels of evaluation

  • Associational. Theta(true_factual,pred_factual) for theta in {F1,precision,recall,accuracy,…}.
  • Interventional. Theta(true_interv1,pred_interv1), F1(true_interv0,pred_interv0) for theta in {F1,precision,recall,accuracy,…}.
  • Counterfactual. Causal reasoning. External validity RAE PNS using both pred_interv1 and pred_interv0. Compositional causal reasoning. Internal consistency and external validity of compositions.

Procedure

  • Causal world model. Define a small, fictional world corresponding to a randomly generated causal graph. This will be the causal world model for the LM to reason over. The structural causal model defining our tiny, fictitious world is comprised of exogenous noise variables, endogenous variables, and causal functions (logical operators and, or).
  • Causal context prompt. Construct a verbal description of the world model. This verbal description — our “causal context prompt” — contains all pertinent details needed for the LM to infer the world model, as well as extraneous details not needed to solve the CCR task.
  • Sampling. Randomly sample exogenous variables and extraneous variables and compute true endogenous variable values. We can sample from observational distribution p or from interventional distributions p'_{0} and p'_{1}. The two interventional distributions can be used to estimate counterfactual measures.
  • Factual query prompts. Construct factual queries using observational data sampled from p, which test for relation extraction and reasoning at the associational level of the PCH.
  • Counterfactual query pairs. Construct paired interventional queries corresponding to p'_{0} and p'_{1}, which together constitute our counterfactual query pairs. These are used to test for reasoning at the interventional and counterfactual rungs of the PCH.
